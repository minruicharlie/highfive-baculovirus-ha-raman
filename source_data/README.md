# Source Data

This directory contains machine-readable non-sensitive source data for the
reported manuscript tables, supplementary tables, selected figure summaries, and
Supplementary Figure S1.

## Contents

- `tables/`: source data for the main tables and supplementary tables.
- `figures/`: aggregate or summary source data for main-text figure panels.
- `supplementary_figure_s1/`: source data used to draw Supplementary Figure S1.

## Withheld Point-Level Files

Point-level source-data files that include offline `reference` values are not
included in the public repository because they would disclose the paired offline
manufacturing/reference records withheld under the data-availability statement.
The public files therefore emphasize aggregate metrics, ablation summaries,
diagnostic summaries, label/count summaries, and online prediction outputs that
do not reveal the withheld paired reference records.
