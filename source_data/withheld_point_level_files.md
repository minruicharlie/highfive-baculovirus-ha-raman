# Withheld Point-Level Files

The following generated point-level files are not included in the public
repository because they contain offline `reference` values and would disclose
the paired offline manufacturing/reference records withheld under the data
availability statement:

- `results/figures/source_data/development_offline_points_source.csv`
- `results/figures/source_data/figure2_via_final_cv_predictions_source.csv`
- `results/figures/source_data/figure3_dim_final_cv_predictions_source.csv`
- `results/figures/source_data/figure4_ha_cv_predictions_source.csv`
- `results/figures/source_data/figure5_independent_test_offline_reference_source.csv`
- `results/figures/source_data/plsr_cv_point_diagnostics_source.csv`
- `results/model_outputs/*/*point_predictions*.csv`

Public replacements are provided as aggregate table source data, figure
diagnostic summaries, ablation summaries, model-output summaries, label/count
summaries, and non-reference online prediction outputs.
